Coolify をインストールする前に、VPS のファイアウォールで以下のポートを開放してください。

## 開放するポート

| ポート | 用途 | 備考 |
| :-- | :-- | :-- |
| 22 | SSH | [VPS セキュリティ](/guides/vps/security/) で設定済みのはず |
| 80 | HTTP | Web トラフィック・Let's Encrypt 検証用 |
| 443 | HTTPS | SSL/TLS 暗号化通信 |
| 8000 | Coolify 管理画面 | インストール直後の初期アクセス用（後で閉じる） |

## 設定方法

VPS プロバイダの管理画面から、上記ポートの **インバウンド（受信）** を許可します。  
プロバイダによって設定画面の名称は異なります（「パケットフィルタ」「ファイアウォール」「セキュリティグループ」など）。

### [[guides/vps/security]] との役割分担

- **VPS セキュリティ（前章）**: ファイアウォールの基本原則と SSH（22）の保護
- **このページ**: Coolify 固有で追加が必要なポート（80 / 443 / 8000）

## 8000 番ポートについて

インストール直後は Coolify 管理画面が **8000 番ポート** で起動します。  
[[guides/coolify/installation|インストール手順]] の完了後、管理画面 URL（ドメインまたは IP）でアクセスできるようになったら **8000 番ポートは閉じてください**。  
以降は 80 / 443 経由で管理画面にアクセスします。

## 次のステップ

ポートの開放が完了したら、[[guides/coolify/installation|インストール手順]] に進みましょう。
