Coolify と Docker を VPS 上で運用するには、ある程度のリソースが必要です。  
このサイトで奨励する条件は以下です。条件を満たすプランを選んでください。

## 最低要件

| 項目 | 要件 | 備考 |
| :-- | :-- | :-- |
| CPU | 2 コア以上 | Docker コンテナの並列実行に必要 |
| メモリ | 4 GB 以上 | Coolify 本体 + アプリ用の余裕 |
| ストレージ | 50 GB 以上 | OS・Docker イメージ・アプリデータ用 |
| OS | Ubuntu 24.04 以降 | Coolify 公式が Ubuntu を推奨 |

## なぜ 4 GB 以上か

Coolify 自体が Docker コンテナとして動作し、デプロイするアプリケーションもコンテナで実行されます。  
メモリが不足すると、インストール中の停止やアプリの不安定な動作につながるため、**4 GB 以上** を目安にしてください。
